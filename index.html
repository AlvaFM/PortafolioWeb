Mis Skills

Echa un vistazo a los lenguajes de programación con los que estoy familiarizado.

  • Durante mi primer semestre, cursé y terminé el aprendizaje de Python.

    Python es un lenguaje de programación interpretado, de alto nivel y fácil de aprender. Es conocido por su sintaxis clara y legible, lo que lo hace ideal para principiantes y profesionales por igual. Python es versátil y se utiliza en una amplia gama de aplicaciones, desde desarrollo web y científico hasta inteligencia artificial y análisis de datos. Su comunidad activa y sus numerosas bibliotecas hacen que sea una opción popular para diversos proyectos de software.

  • Durante mi segundo semestre, cursé y terminé el aprendizaje del lenguaje de programación Java.

    Java es un lenguaje de programación orientado a objetos, portable y de propósito general. Destaca por su seguridad, portabilidad y amplio uso en una variedad de aplicaciones, desde desarrollo web hasta aplicaciones móviles y sistemas embebidos. Su lema "Write Once, Run Anywhere" resalta su capacidad para ejecutarse en diferentes plataformas con una sola implementación del código.

  • Actualmente me encuentro cursando un curso de HTML/CSS.

    HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Define la estructura y el contenido de una página web mediante etiquetas, que luego son interpretadas por los navegadores web para mostrar el contenido al usuario. CSS (Cascading Style Sheets) es un lenguaje utilizado para controlar el aspecto y el diseño de las páginas web escritas en HTML. Permite aplicar estilos como colores, fuentes, márgenes, y más, para mejorar la apariencia y la presentación de un sitio web. CSS funciona de manera complementaria con HTML para crear páginas web visualmente atractivas y funcionales.

  • Actualmente me encuentro realizando mi curso de JavaScript.

    JavaScript (JS) es un lenguaje de programación de alto nivel, interpretado y orientado a objetos. Es conocido como el lenguaje de la web, ya que se utiliza principalmente para agregar interactividad y dinamismo a las páginas web. JavaScript se ejecuta en el navegador del cliente y permite manipular el contenido HTML, responder a eventos del usuario, realizar peticiones al servidor y mucho más. Es fundamental en el desarrollo web moderno, y su versatilidad lo hace útil tanto en el frontend como en el backend, gracias a entornos como Node.js.
